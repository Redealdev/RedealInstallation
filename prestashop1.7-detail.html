Redeal Prestashop 1.7 plugin installation

Follow the steps

  1. Download the plugin here.
  2. First, go to admin panel of your Prestashop1.7 store.
  3. Then click on "Modules" link in the left sidebar.
  4. Click "Upload a Module" link and select as redeal.zip file to install new module in the top right of the admin panel.
  5. After upload the file successfully, You need to find “Redeal” module and click on install button in module list and install it. See here
  6. After successfully installation you will see Redeal Configuration Page.
  7. Do the configuration setting as per your requirement.
  8. After that you can place order and can see Redeal popup in order confirmation page. See here.
  9. Create Google tag manager account and a GTM code from here
  10. Copy the GTM Code from here
  11. Add your GTM Code in Redeal Configurations from here
