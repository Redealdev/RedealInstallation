RedealMagento 1.x Plugin Installation Guide

Steps are as below.

  1. Please Extract zip file on Magento root folder.
  2. ogin to Magento admin and if you are already logged in then refresh the magento cache.
  3. Go to System > Configuration >RedealReferralmarketing.On left side you can see Tab "RedealReferralmarketing". Click on RedealReferralmarketing Options.
  4. Right side you can see the options.
  5. To enable the extension Select "Enable" ->"Yes".
  6. To use Google Tag Manager enter Google tag manager container ID. You can get container id from your google tag manager account. How to get Container ID: https://support.google.com/tagmanager/answer/6103696?hl=en
  7. Clear the Magento cache.

That's it!
