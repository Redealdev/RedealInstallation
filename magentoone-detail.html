Redeal Magento 1.x plugin installation

Steps are as below.

  1. Download the plugin here.
  2. Extract zip file and rename folder name to "app"
  3. Copy app folder and Paste it on root folder of your web server.
  4. Login to Magento admin and if you are already logged in then refresh the magento cache.
  5. Go to System > Configuration >RedealReferralmarketing. On left side you can see Tab RedealReferralmarketing". Click on RedealReferralmarketing Options.
  6. Clear the Magento cache.
