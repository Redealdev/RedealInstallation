Redeal Opencart 1.5 plugin installation

Follow the steps

  1. First, Unzip/Extract the RedealPlugin-Opencart-app-15-1.0.0.zip file here.
  2. Then, Copy the contents of the "upload" directory into the root of your openCart installation.
    Note: Do not copy the upload folder itself, just the files & directories inside.
  3. You must need to install vqmod extension for this module.
  4. You can download vqmod from this url : here.
  5. After installing vqmod extension go to the root of the opencart.
  6. Place redeal.xml file into "vqmod/xml" folder. ( redeal.xml file is available in redeal.zip file)
  7. Then, Go to admin panel of your Opencart1.5 store.
  8. Go to the "Extension->Modules" from top menu. See below image here.
  9. If redeal module is proper installed then It will display in modules here.
  10. Then, Install redeal module. here.
  11. Edit redeal module and add GTM Id and enable redeal module and save setting. here.
  12. Create Google tag manager code from here
  13. Copy your GTM Code from here
  14. On Dashboard, You will see the GTM(Google Tag Manager) field where you can add GTM code and click on save button.
