RedealMagento 2.x Plugin Installation Guide

Steps are as below.

  1. Please Extract zip file on root folder.
  2. Please Check if previous version exists or not. app->code->Redeal->Referralmarketing
  3. Run this Commands.
    • ● php bin/magentosetup:upgrade
    • ● php bin/magentosetup:static-content:deploy
    • ● php bin/magentocache:flush
  4. Create Google tag manager code from => https://drive.google.com/file/d/1i5-pDYy5h92_5MBXY0im-D4OImRHMYqe/view?usp=drivesdk
  5. GTM Code => https://drive.google.com/file/d/1pX-kkUvxbSMXs7ZxouT1pCpXtPZ8vIdV/view?usp=drivesdk
  6. Add GTM Code in Store-> Setting -> Configuration ->REDEAL REFERRALMARKETING -> https://drive.google.com/file/d/1YYCss_TtdlScpwTGRG2WEStEnPGLsRfg/view?usp=drivesdk

That's it!
