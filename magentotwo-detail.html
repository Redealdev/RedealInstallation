Magento 2.x

Redeal Magento 2.x plugin installation
  1. Download the plugin here.
  2. Extract zip file on root folder on your web server.
  3. Check if previous version exists or not. app -> code -> Redeal -> Referralmarketing
  4. Run these commands:
    • php bin/magentosetup:upgrade
    • php bin/magentosetup:static-content:deploy
    • php bin/magentocache:flush
  5. Create Google tag manager code from -> https://tinyurl.com/yd2udnwz
  6. Copy your GTM Code from -> https://tinyurl.com/y8nglcc7
  7. Add your GTM Code in Store -> Setting -> Configuration -> REDEAL REFERRALMARKETING ->https://tinyurl.com/ybqxrfjy
