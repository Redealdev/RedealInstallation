Redeal WooCommerce Plugin Installation Guide

Steps are as below.

  1. Download plugin from this link
  2. Login now from admin panel using this link - http://www.exampledomain.com/wp-admin
  3. Go to this path for upload plugin file from wp-admin. Path:- wp-admin -> Plugins->Add New->Upload Plugins->Choose file (Plugin zip file from your computer).
  4. After successfully installation, you can see the configuration option in Left menu ("Redeal Configurations"). Click on it.
  5. Do the configuration as per your requirement.
  6. Create Google tag manager code from => https://drive.google.com/file/d/1i5-pDYy5h92_5MBXY0im-D4OImRHMYqe/view?usp=drivesdk
  7. GTM Code = > https://drive.google.com/file/d/1pX-kkUvxbSMXs7ZxouT1pCpXtPZ8vIdV/view?usp=drivesdk
  8. Add GTM Code in Redeal Configurations https://drive.google.com/file/d/1i5-pDYy5h92_5MBXY0im-D4OImRHMYqe/view?usp=drivesdk

That's it!
