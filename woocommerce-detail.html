Redeal Woocommerce plugin installation

Follow the steps

  1. Download the plugin here.
  2. Extract zip file and rename it "custom-plugin"
  3. Login in to your admin panel. Eg http://www.exampledomain.com/wp-admin
  4. If you are installing plugin first time than upload plugin file in wp-admin following the path: wp-admin -> Plugins-> Add New -> Upload Plugins -> Choose the plugin zip file from your computer.
  5. If You have already installed plugin than copy "custom-plugin" folder and paste it on this path : wp-content->plugins.
  6. After successfully installation, You need to activate plugin from plugin lists.
  7. After that you can see the configuration option in the left menu Redeal Configurations. Click on it.
  8. Do the configuration as per your requirement.
  9. Create Google tag manager account and a GTM code from here
  10. Copy the GTM Code from here
  11. Add your GTM Code in Redeal Configurations from here
