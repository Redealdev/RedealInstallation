Redeal Woocommerce plugin installation

Follow the steps

  1. Download the plugin here.
  2. Login in to your admin panel. Eg http://www.exampledomain.com/wp-admin
  3. Upload plugin file in wp-admin following the path: wp-admin -> Plugins-> Add New -> Upload Plugins -> Choose the plugin zip file from your computer.
  4. After successfully installation, you can see the configuration option in the left menu Redeal Configurations. Click on it.
  5. Do the configuration as per your requirement.
  6. Create Google tag manager account and a GTM code from -> https://tinyurl.com/yd2udnwz
  7. Copy the GTM Code -> https://tinyurl.com/y8nglcc7
  8. Add your GTM Code in Redeal Configurations https://tinyurl.com/ybqxrfjy
