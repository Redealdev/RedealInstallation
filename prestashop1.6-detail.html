Redeal Prestashop 1.6 plugin installation

Follow the steps

  1. Download the plugin here.
  2. First, go to admin panel of your Prestashop1.6 store.
  3. Click on "Module & Services" from Left Menu Sidebar.
  4. If you are installing new module
    • Click "Add a new Module" link to install news module in the top right of the admin panel here.
    • After click on it you see the box "ADD NEW MODULE" box in which you need to upload the "redeal.zip" file and click on "upload this module" button. And after is you can see here
    • After upload the file successfully, You need to find “Redeal” module in module list and install it here.
  5. If you have already installed a module
    • You need to delete existing module here
    • After need to add new module here
    • After click on it you see the box "ADD NEW MODULE" box in which you need to upload the "redeal.zip" file and click on "upload this module" button. And after is you can see here
    • After upload the file successfully, You need to find “Redeal” module in module list and install it.
  6. After successfully installation you will see Redeal Configuration Page here.
  7. Do the configuration setting as per your requirement.
  8. After that you can place order and can see Redeal popup in order confirmation page. See here.
  9. Create Google tag manager account and a GTM code from here
  10. Copy the GTM Code from here
  11. Add your GTM Code in Redeal Configurations from here
