Redeal Prestashop 1.6 plugin installation

Follow the steps

  1. Download the plugin here.
  2. Unzip/Extract the zip file which you have downloaded and rename the folder inside the zip file to "redeal"
  3. Now, Create a zip of this redeal folder/directory with the same name "redeal"
  4. Go to admin panel of your Prestashop1.6 store.
  5. Click on "Module & Services" from Left Menu Sidebar.See here
  6. Click "Add a new Module" link in the top right of the admin panel to upload the "redeal" module. See here
  7. After clicking on it you see the box "ADD NEW MODULE" in which you need to upload the "redeal" zip file and click on "upload this module" button. See here
  8. Once the zip file is uploaded successfully, you will find “Redeal” module in Prestashopmodule list. See here
  9. If you already have “redeal” module installed, it will update the existing module and its version. If it is not already installed, it will create a new module.
  10. Click on install/configure button to configure this module. See here
  11. After successfully installation you will see Redeal Configuration Page here.
  12. Do the configuration setting as per your requirement.
  13. After that you can place order and can see Redeal popup in order confirmation page. See here.
